# algorithms
Мини программы по алгоритмам.  

Circular Linked List - Создать кольцевой список. Удалить каждый K -ый элемент. В списке находятся символьные данные. Результирующий список вывести на экран.  

double ended queue.cpp - Создать две двунаправленные очереди с целыми числами. Найти в первой очереди сумму элементов, которых нет в другой очереди. Вывести на экран полученный результат.
